# Vue.js로 만든 ToDo List

## 주요 기능
- 목록 전체 삭제
- 목록별 삭제, 완료
- 목록을 버튼, 엔터로 추가 가능
- 아이폰 사파리 기준 공유->홈 화면 바로 가기 추가 시 앱처럼 사용 가능
(manifest.json, index.html(<link rel="manifest" href="/manifest.json"> 코드 추가))
